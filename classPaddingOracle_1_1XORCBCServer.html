Detailed Description

A TCP-server providing an interface against which to demonstrate the padding oracle attack.

The padding oracle protocol works as follows:

  1. client: 16 bytes ciphertext
  2. while :
    1. client: 2 bytes Q block count ( ) (little endian)
    2. client: times
      • 16 bytes Q block
    3. server: times:
      • 1 if Q block resulted in correct padding
      • 0 otherwise
